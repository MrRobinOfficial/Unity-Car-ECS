<h1 align="center">Unity Car ECS [Unity Engine]</h1>

*A car game built in Unity using the Entity Component System (ECS) and optimized C# code for maximum performance. This game is designed to showcase the power and efficiency of ECS and optimized code in creating a high-performance car game.*

<div align="center">

This repo is built by using [Unity DOTS](https://unity.com/dots).
